# Maze Intelligence

## Integrantes

- Sara Castrillón
- Alejandro Cadavid

## Descripcion

Maze Intelligence es un videojuego en el lenguaje Jack. El objetivo del juego es guiar al jugador a través de un laberinto hasta encontrar la salida, mientras evita ser atrapado por un enemigo que utiliza un algoritmo de búsqueda BFS (Breadth First Search) para perseguirlo.

El proyecto aplica conceptos de programación orientada a objetos, estructuras bidimensionales, algoritmos de búsqueda, manejo de memoria y representación gráfica utilizando las librerías de Hack

## Características 

- Laberinto basado en una matriz de 16x16
- Movimiento del jugador por flechas del teclado
- Detección de choque contra muros
- Enemigo con persecución inteligente utilizando BFS
- Sistema de victoria y derrota
- Reinicio y salida del juego
- Representación gráfica usando la clase Screen
